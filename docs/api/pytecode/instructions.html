pytecode.instructions

Data types representing JVM bytecode instruction operands.

Provides dataclasses and enums that model the operand formats for each JVM instruction as defined in the JVM specification (JVMS §6.5). Each opcode is mapped to an InsnInfoType member whose associated InsnInfo subclass describes the shape of its operands.
